Furniture Planner — How to Open (Mac)
======================================

1. Unzip this folder.
   - Double-click FurniturePlanner.zip in Finder.
   - A new folder called "FurniturePlanner" will appear.

2. Open the folder.

3. Double-click the file named:  planner.html
   - It will open in your web browser (Safari or Chrome).

4. When the page opens, it will ask you to select a CSV file.
   - Click the "Choose File" button.
   - Pick the file named:  furniture_proposals.csv
     (it's in the same folder you just unzipped)

5. That's it — the planner will load with all items, prices, and images.

Tips
----
- If images don't show up in Safari, try opening planner.html in Chrome instead.
- To re-open later, just double-click planner.html again. Your CSV stays in the folder.
- Nothing is installed on your Mac. To remove, just drag the folder to the Trash.
